# Cara Menjalankan

1. Clone Repository
Jalankan perintah berikut di terminal:

git clone https://github.com/anjat6262/Larisdy.com.git

2. Masuk ke Folder Project
cd Larisdy.com

3. Buka Folder Source Code
Masuk ke folder src

4. Jalankan Aplikasi
Buka file index.html menggunakan browser (Chrome/Edge/Firefox)

5. (Opsional - jika pakai Live Server)
Jika menggunakan VS Code:
- Install extension Live Server
- Klik kanan index.html
- Pilih "Open with Live Server"
